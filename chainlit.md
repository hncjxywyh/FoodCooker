# 欢迎使用 FoodCooker！🍳🤖

你好，开发者！👋 欢迎使用 FoodCooker —— 一个基于 LangChain + RAG 的个性化食谱推荐助手。

## 功能说明 📋

- 根据你现有的食材推荐美味菜谱
- 根据你的饮食偏好（过敏、口味、菜系）定制食谱
- 计算营养成分，生成购物清单

## 使用方法 🍽️

1. 告诉我你现有的食材（如：鸡蛋、西红柿）
2. 告诉我你的饮食偏好（如：不吃辣、清淡）
3. 我会为你推荐并定制专属食谱！

## 常见问题 💡

**Q: 如何更改语言偏好？**
直接在对话中告诉我你的偏好即可，我会记住并在后续推荐中考虑。

**Q: 食材不够怎么办？**
我会自动生成购物清单，告诉你还需要购买哪些食材。

祝你烹饪愉快！💻😊
